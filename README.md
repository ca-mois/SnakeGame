## 1. Cài đặt
### 1.1. Phiên bản Python  
``` 3.8.2```

### 1.2. Cài đặt thư viện
- Cài các thư viện trong requirements.txt: 
```bash
pip install -r requirements.txt
```
## 2. Chạy chương trình:  
Chạy file ```SnakeGame.py```  
Để vào game => Nhấn phím a  
Để thoát game/camera => Nhấn phím q
